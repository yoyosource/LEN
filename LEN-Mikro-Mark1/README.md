# LEN Mikro mk1 (LixEntNow Mikro Mark 1)
Finished building on 29 March of 2021

## Instruction Set
- a+b -> call a + cell b
- a-b -> cell a - cell b
- rotate -> rotate to next cell
- input -> read input and put value cell c
- b>c -> reset cell a

## Stats
- Width: 10
- Height: 3
- Depth: 10
- AU (Arithmetic Unit)
- LU (Logic Unit)
- Ram: 4 values
